# QrCodeApp #
Small app for generating and reading QR codes on Windows Desktop.

![preview][1]

### Frameworks ###
- .NET 4.7
- WPF
- MVVM: [Prism][2]
- Style: [MahApps.Metro][3]
- QR Code Reader/Generator: [zxing][4]
